
أريد تخصيص واجهة متكاملة لتطبيق سطح مكتب إسلامي متقدم اسمه "الكتاب المبين".
تفاصيل الواجهة المطلوبة:
شاشة الدخول (Login Screen):
تحتوي على حقلين: Username, Password.
زر دخول واضح باسم "LOG IN".
أيقونات صغيرة لتسجيل الدخول عبر منصات التواصل الاجتماعي.
في الأعلى، لوجو التطبيق بخط عربي واضح ومميز، النص: "الكتاب المبين".
الداشبور (Dashboard):
تصميم واضح وأنيق بخلفية خضراء داكنة تتدرج إلى ألوان فاتحة.
تقسيمات واضحة تشمل:
مواقيت الصلاة القادمة بشكل بارز مع مؤقت تنازلي.
قسم للأذكار اليومية بتفاعل سهل وواضح.
قسم خاص بالتفسير وآيات اليوم، يعرض آية واحدة مع تفسير مختصر وزر لعرض التفسير الكامل.
شريط التنقل (Sidebar Navigation):
أيقونات واضحة بألوان متناغمة مع التصميم العام.
أقسام تشمل:
الصفحة الرئيسية (Home)
القرآن الكريم (Quran)
مكتبة التفسير (Tafseer Library)
الأذكار اليومية (Daily Athkar)
التقويم الهجري (Islamic Calendar)
إدارة الحساب (Account Management)
الإعدادات (Settings)
قسم خاص بالأطفال (Kids Section):
تصميم مبهج وجذاب، يحوي أنشطة تفاعلية لتعليم الأطفال مبادئ الدين والصلاة وحفظ القرآن.
صور ورسومات جذابة للأطفال مستوحاة من الثقافة الإسلامية.
قسم VIP خاص بالأعضاء المميزين:
محتوى حصري (محاضرات، خطط حفظ، جلسات تعليمية مباشرة).
بطاقة عضوية رقمية واضحة ومميزة.
قسم إدارة خاص بالأئمة والمعلمين:
لوحة تحكم لإدارة الطلاب.
متابعة الخطط التعليمية.
تقييم الأداء وتقديم النصائح.

الواجهة تكون سهلة وواضحة، مع دعم كامل للغتين العربية والإنجليزية.
الخلفية:
لون أخضر داكن (Islamic dark green).
تدرج سفلي نحو الذهبي (Gold-Green gradient).
شفافية خفيفة لصورة داخلية للمسجد النبوي أو الطراز المعماري الإسلامي (مثل خلفية القبة أو الأعمدة).
الشعار الرئيسي (Logo Box):
داخل إطار ذهبي مربع الزوايا (Rounded square frame).
مكتوب بخط عربي جمالي: "الكتاب المبين" (بالكاف).
تحته شريط ذهبي يحتوي العبارة: "عِلْمٌ يُنتَفَعُ بِه" أو أي عبارة قرآنية (مثل: ﴿أَلَمْ يَعْلَمْ بِأَنَّ اللَّهَ يَرَى﴾).
حقول الدخول:
حقل Enter User Name (Text Input Field).
حقل Enter Password (Password Input Field).
زر LOG IN بلون ذهبي محاط بظل ناعم.
أيقونات تسجيل الدخول السريع (Google – Facebook – Apple).
